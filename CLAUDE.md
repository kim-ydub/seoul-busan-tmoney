# 서울→부산 시내버스 여행 지도
_최종 업데이트: 2026-03-16 16:52:08 (KST)_

## 프로젝트 개요
2026년 2월 28일 ~ 3월 5일 (6박 7일) 동안 서울에서 부산까지
시내버스와 광역전철만으로 여행한 경로를 한국 지도 위에 시각화한 프로젝트.

## 배포 정보
- **GitHub Repository**: https://github.com/kim-ydub/seoul-busan-tmoney
- **GitHub Pages URL**: https://kim-ydub.github.io/seoul-busan-tmoney/
- **배포 방법**: `git add . && git commit -m "메시지" && git push`

## 기술 스택
- **지도**: Leaflet.js (OpenStreetMap 타일, 세피아 필터 적용)
- **경로**: 하드코딩된 STOPS 좌표 간 직선 렌더링 (외부 API 없음)
- **폰트**: 시스템 폰트 우선 (Apple SD Gothic Neo → Malgun Gothic → Noto Sans KR)
- **언어**: HTML / CSS / JavaScript (단일 파일)

## 데이터 구조 (index.html 내부)
- **STOPS**: 정류장/역 이름 → 위경도 좌표 매핑 객체 (수동 하드코딩)
- **SEGMENTS**: 34개 이동 구간 배열

- **DAY_COLORS**: 일차별 색상 (1일차~7일차)

## 모바일 레이아웃 (768px 이하)
- 사이드바 → `position: fixed` 하단 드로어로 전환
- 기본 상태: 드로어 32vh 노출 (지도 상단 70% 노출)
- 스와이프 업: 드로어 88vh까지 확장
- 스와이프 다운: 32vh 복귀
- 드로어 상단: 회색 핸들 바 + 일차 필터 버튼(가로 스크롤)
- 터치 이벤트: 핸들 바 / 사이드바 헤더에서만 드래그 인식 (구간 목록은 별도 스크롤)
- 방향 감지: 수평 스와이프 시 드로어 개입 없이 가로 스크롤 허용, 수직 스와이프만 드로어 동작
- 헤더: 48px으로 축소, 통계 숨김

### 3. 컴포넌트

#### 일차 필터 버튼 (Pill 토글)
- 기본: `--neutral-0` 배경, `--neutral-300` 테두리, `--neutral-700` 텍스트, `border-radius: 20px`
- 활성: `--green-2` 배경, 흰색 텍스트 (일차별 색상 구분 없이 통일)

#### 구간 타입 배지
- 버스: `--green-5` 배경 + `--green-1` 텍스트
- 전철/지하철: `#dbeafe` 배경 + `#1e3a8a` 텍스트

#### 헤더
- 배경: `--green-1`, 타이틀: `--green-5`, 통계 수치: `--green-4`

#### 팝업 / 토스트
- 배경: `--green-1`, 텍스트: `--green-5`, 강조: `--green-4`

### 4. 디자인 원칙

1. **그린 중심**: 모든 강조와 액션은 브랜드 그린 팔레트 안에서 해결한다. 파란색 계열 강조는 지양한다.
2. **배경 계층 3단계**: 페이지(`--neutral-100`) → 컨텐츠(`--neutral-0`) → 인라인 강조(`--neutral-100` 또는 `--green-5`)
3. **텍스트 위계**: `--neutral-900` → `700` → `500` → `300` 순으로 중요도 구분
4. **그림자 최소화**: 플로팅 요소(팝업, 드로어)에만 그림자 사용
5. **시스템 폰트 우선**: 외부 폰트 없이 시스템 폰트로 먼저 렌더링, Noto Sans KR은 폴백
6. **의미 기반 색상**: 같은 항목은 항상 같은 색 (일차별 지도 색상은 가독성 우선 예외 허용)
