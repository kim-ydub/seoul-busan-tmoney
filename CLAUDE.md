## 프로젝트 개요

서울→부산 시내버스 여행기 (2026.02.28 ~ 2026.03.05, 6박 7일)를 인터랙티브 지도로 시각화한 단일 HTML 파일 프로젝트.

- **파일 구성**: `index.html` 단일 파일 (CSS, JS 인라인 포함)
- **지도 라이브러리**: Leaflet.js 1.9.4 (OpenStreetMap 타일)
- **폰트**: Noto Serif KR, DM Mono (Google Fonts)
- **저장소**: https://github.com/kim-ydub/seoul-busan-tmoney

---

## 데이터 구조

### STOPS
정류장 이름 → `[lat, lng]` 좌표로 매핑된 객체. 모두 수동 하드코딩.

### SEGMENTS
34개 이동 구간 배열. 각 구간은 아래 필드를 가짐:
- `day`: 일차 (1~7)
- `transfer`: 환승 번호
- `type`: `'bus'` | `'rail'`
- `route`: 노선명
- `from` / `to`: STOPS 키
- `dep` / `arr`: 출발/도착 시각
- `cost`: 요금 (원, 환승 무료는 0)

### DAY_COLORS
일차별 폴리라인 색상 (7가지).

## 지도 렌더링

- 타일에 `saturate(0%) brightness(88%) sepia(14%)` 필터 적용 (세피아 톤)
- 폴리라인: 버스=실선, 전철/지하철=점선(`8,5`)
- 마커: 출발지(주황 테두리), 도착지(빨간 원), 중간 정류장(일차 색상 테두리)
- OSRM 도로 경로 API **사용 안 함** — STOPS 좌표 간 직선으로 렌더링 (즉시 로드)

---

## 주요 기능

- **일차별 필터**: 사이드바 상단 버튼으로 1~7일차 또는 전체 표시
- **구간 클릭**: 사이드바 항목 클릭 시 해당 구간으로 지도 줌
- **범례**: 실선=시내버스, 점선=광역전철·지하철
- **헤더 통계**: 총 교통비(JS로 계산), 총 구간 수(34), 경유 도시 수(7)
